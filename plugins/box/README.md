Box helps you quickly find, reference, and summarize documents from your Box workspace.

## What It Can Do
- Searches Box files and folders from your connected workspace.
- Pulls relevant documents into task context for summarization and synthesis.
- Supports document-grounded answers for planning and implementation work.

## Why Use It
- Keeps decisions grounded in source docs instead of memory.
- Saves time hunting for files across large Box workspaces.
- Makes knowledge retrieval faster during active coding sessions.

## How It Works
1. Connect your Box app integration in Codex.
2. Ask the plugin to find files or summarize specific documents.
3. Use extracted context to drive implementation or communication tasks.

## Quick Examples
- `Find a Box file and summarize the key points`

## Requirements
- A connected Box app integration with access to your target files.
