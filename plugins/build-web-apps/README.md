Build Web Apps helps you design, build, deploy, and scale modern web applications end to end.

## What It Can Do
- Improves UI/UX and frontend quality using web design and React best-practice workflows.
- Supports deployment and production workflows via Vercel tooling.
- Guides payments and database decisions with Stripe and Supabase/Postgres expertise.

## Why Use It
- Unifies frontend, backend, data, and deployment guidance in one plugin.
- Reduces context switching between separate docs/tools during feature delivery.
- Helps teams move from prototype to production with fewer blind spots.

## How It Works
1. Use the right skill for the job (UI review, React refactor, deploy, payments, DB tuning).
2. Leverage built-in MCP connections for Vercel, Stripe, and Supabase when live context is needed.
3. Apply changes in-repo and validate with your normal test/deploy checks.

## Quick Examples
- `Review this app's UI, improve the React implementation, wire up payments or database changes, and help deploy it`

## Requirements
- Project access in Codex with relevant source code.
- Optional: connected Vercel/Stripe/Supabase accounts for MCP-backed actions.
