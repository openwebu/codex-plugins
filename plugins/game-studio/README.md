Game Studio helps you plan, prototype, and ship browser games across 2D and 3D stacks.

## What It Can Do
- Plans and prototypes browser games across 2D and 3D stacks.
- Supports gameplay loop design, UI, asset pipelines, and playtest workflows.
- Covers common web game frameworks like Phaser and Three.js-based stacks.

## Why Use It
- Gives you a practical structure from concept to playable build.
- Helps avoid early architecture mistakes in game projects.
- Accelerates iteration on mechanics, visuals, and performance.

## How It Works
1. Start with game foundations and core-loop planning skills.
2. Pick a framework-focused path (2D or 3D) and implement features incrementally.
3. Use playtest-oriented workflows to gather evidence and tune gameplay.

## Quick Examples
- `Design a browser game and plan the core loop`

## Requirements
- A JavaScript/TypeScript web game project workspace.
- Framework/tooling setup for your chosen stack (for example Phaser or Three.js).
