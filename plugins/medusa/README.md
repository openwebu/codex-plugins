## What It Can Do
- Builds Medusa backend modules, workflows, API routes, and data models.
- Supports admin dashboard customizations and storefront integrations.
- Includes MedusaDocs MCP support for up-to-date implementation guidance.

## Why Use It
- Covers the full Medusa stack in one plugin instead of piecing workflows together.
- Improves correctness by combining architecture patterns with docs-backed references.
- Accelerates ecommerce feature delivery across backend and frontend teams.

## How It Works
1. Choose the matching Medusa skill (backend, admin, storefront, migrations, learning).
2. Use MedusaDocs MCP to verify patterns and APIs before implementation.
3. Implement incrementally and validate behavior in your Medusa project.
