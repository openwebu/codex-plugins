Netlify helps you deploy projects, manage previews, and ship production releases with confidence.

## Top Skills
- `netlify-deploy`

## What It Can Do
- Deploys projects to Netlify with preview and production release workflows.
- Helps inspect deployment state and identify release blockers.
- Pairs app integration access with deployment-oriented skills.

## Why Use It
- Simplifies going live without leaving your coding workflow.
- Improves deployment confidence through repeatable release steps.
- Reduces downtime caused by manual deploy process errors.

## How It Works
1. Connect Netlify in Codex.
2. Run deployment workflows to generate preview or production releases.
3. Review deployment output, fix blockers, and redeploy as needed.
