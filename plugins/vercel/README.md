Vercel helps you build, deploy, operate, and optimize applications across the full Vercel ecosystem.

## What It Can Do
- Covers the full Vercel ecosystem: deployments, runtime behavior, observability, and platform services.
- Includes extensive skills for Next.js, AI SDK, auth, storage, CI/CD, and performance.
- Connects to the official Vercel MCP server for docs and live project context.

## Why Use It
- Gives one consistent workflow for shipping and operating Vercel-hosted apps.
- Speeds up troubleshooting with integrated runtime/deployment guidance.
- Helps teams make better architecture choices across frontend, backend, and AI features.

## How It Works
1. Start with the skill that matches your current task (deploy, debug, optimize, integrate).
2. Use Vercel MCP for up-to-date docs and environment-aware project insights.
3. Implement, verify, and iterate in the same loop until production-ready.

## Quick Examples
- `Audit this repo for Vercel deployment risks`
- `Which Vercel tools fit this app best`
- `Help wire Vercel MCP into this workflow`

## Requirements
- Vercel account/project access and OAuth authorization for MCP.
- A codebase targeting Vercel deployment workflows.
